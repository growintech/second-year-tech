Step 2 — The Initial Prompt

Fix the Replit Prompt

The prompt in the starter code will make Replit generate a React app. You need to rewrite it so the result is a plain HTML/CSS/JS website.

  • Find the <pre id="prompt"> element in the HTML
  • Replace its content with a correct initial prompt
  • The prompt must mention HTML, CSS, and JavaScript
  • It must say No frameworks
  • It must name the three files: index.html, style.css, script.js
  • It must include the /public folder for CSS and JS
Hint 1: Start with "Create a simple static website project..." and remove any mention of React or animations.
Hint 2: Add a line: "The website itself must use ONLY plain HTML, CSS and JavaScript. No frameworks."
Hint 3: Add a line naming the three files, then "Put style and script in the /public folder."
