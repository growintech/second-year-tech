Step 4 — Inspect & Download

Complete the Generated File

The starter code is a sample generated index.html. Two things are missing from the <head> — add them to make the file complete.

  • Add a <title> tag with your website name
  • Add <link rel="stylesheet" href="style.css"> to load the styles

Click Run to preview, then Check to validate.

Hint 1: Both tags go inside <head>, below the existing <meta> tags.
Hint 2: A title tag looks like: <title>My Site</title> — replace "My Site" with any name.
Hint 3: The stylesheet link is: <link rel="stylesheet" href="style.css"> — it goes in <head>, not <body>.
