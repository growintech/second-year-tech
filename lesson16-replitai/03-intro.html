Build in Replit

Use this page as a reference while you work. Your goal: a JavaScript recap website to study before the final test.

The Replit interface

Once your project is created, Replit shows three panels:

💬 Chat
  • Type your instructions here
  • Send the initial prompt first
  • Then send follow-up requests
📁 Workspace
  • File tree (left sidebar)
  • Code editor (center)
  • Open and edit files here
🖥 Preview
  • Live preview of your site
  • Refreshes automatically
  • Check your changes here

How to start — step by step

1
Create a new Repl — click the + button and select a project type. Choose anything; the initial prompt will override the default setup.
2
Paste the initial prompt in the chat panel and send it. Wait for the agent to finish — this may take a minute.
3
Check the file tree. You should find index.html, style.css, and script.js. If the structure looks wrong, paste the initial prompt again.
4
Send a second prompt describing your recap website content. Be specific — the clearer your instructions, the better the result.
5
Read the generated code. Open index.html, style.css, and script.js. Make sure you understand what each file does. Edit directly if needed.

Ideas for your recap website

Choose one or combine a few. Keep the code simple and understandable.

🧠 Data Types Quiz Multiple-choice questions about numbers, strings, booleans, and null/undefined.
🔀 Conditions Game A small game where the player picks the correct result of an if/else expression.
📋 JSON Explorer A page that displays a JSON object and lets you click to inspect each field.
📝 Functions Glossary A list of JavaScript concepts with short explanations and interactive examples.
💡
If the agent generates something complex — delete it and send the initial prompt again. It is faster to start over than to fix complex generated code.
