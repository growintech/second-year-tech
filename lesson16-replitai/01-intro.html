What is a Coding Agent?

Understand how AI coding agents differ from chatbots — and why Replit needs special instructions.

You've already seen AI tools that answer questions. A coding agent goes further: it reads your project files, writes new ones, and runs commands — all inside a real development environment.

💬 Chatbot

  • Answers questions in text
  • Can suggest code snippets
  • Does not touch your files
  • You copy-paste the output

🤖 Coding Agent

  • Reads and writes project files
  • Creates folders and structure
  • Runs commands, installs tools
  • Shows a live preview directly

The developer's role does not disappear. Your job is still to give clear instructions, read the generated code, and check that it makes sense.

What is Replit?

Replit is a browser-based development environment with a built-in AI coding agent. By default it generates projects using modern frameworks — things like React or Node.js — which are powerful but complex.

⚠
For cambi qualcosa course we only use plain HTML, CSS, and JavaScript. Without a clear initial prompt, Replit will generate code you can't understand yet.

In the next step you'll learn how to write a setup prompt that constrains Replit to produce simple, readable files.
