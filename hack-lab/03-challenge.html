Hidden Element

The IT team left something on MegaCorp's staff page. It's hidden — but still there.

🎯
Open DevTools (F12), go to the Elements tab, and look through the HTML. You're looking for an element that has display:none or visibility:hidden in its style.

In the Elements tab, you can click on any element to see its styles. You can also use Ctrl+F inside the Elements panel to search for text — try searching for "admin" or "GIT{". Look for an element with display:none.

🚩 Submit Flag

🔓 Challenge solved!

display:none hides elements visually, but they remain fully in the HTML sent to the browser. Anyone with DevTools can see them, copy their content, or make them visible with one click. This is a common mistake for hiding premium content, admin features, or sensitive data.

The lesson: Hidden elements are not secret elements. Never use CSS to "hide" sensitive information.
