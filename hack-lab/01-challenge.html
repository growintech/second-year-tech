View Source

MegaCorp forgot to remove a note from their About page. Find it in the HTML.

🎯
Every browser can show you the raw HTML. Try right-clicking the page and selecting View Page Source, or press Ctrl+U / Cmd+U. The flag is hiding in an HTML comment.

About MegaCorp

We build enterprise software solutions for the modern world. Founded in 2018, our team of 200+ engineers ships products used by millions.

Our mission: move fast, break things, and never read the source code.

HTML comments are written like <!-- this is a comment -->. They're invisible on the page but visible in the source. Try Ctrl+U (Windows) or Cmd+U (Mac) to open View Source, then look near the top of the browser-body section.

🔓 Challenge solved!

HTML comments are visible to anyone who views the page source. Developers sometimes leave credentials, API keys, internal notes, or staging URLs in comments — and forget to remove them before going live. This has caused real data breaches.

The lesson: Never put secrets, credentials, or internal notes in HTML comments.
