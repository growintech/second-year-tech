How it works

Each challenge = a target + a flag

You'll see a fake MegaCorp website. Your job is to find the hidden flag — a string in the format GIT{...} — using only your browser's built-in tools. No special software needed.

Use DevTools — that's the point

Open DevTools with F12 (Windows/Linux) or Cmd+Option+I (Mac). The Elements, Console, Application, and Sources tabs are your tools.

Stuck? Use the hint

Every challenge has a hint button. It won't give you the flag — but it'll point you in the right direction. Using a hint is tracked in your results.
