JS Console

A developer left a debug function in MegaCorp's error page. Call it from the console to get the output.

🎯
Open DevTools (F12), go to the Console tab. Type the function name and press Enter to call it. Look at the page source (Ctrl+U / Cmd+U) to find the function name.

Every JavaScript file sent to the browser is public. Debug functions, internal endpoints, API keys, and hardcoded logic are visible to anyone who reads the source. Many real-world breaches have started with a developer forgetting to remove a console.log or debug helper.

The lesson: There is no such thing as client-side secret JavaScript. Everything you send to the browser can be read.
