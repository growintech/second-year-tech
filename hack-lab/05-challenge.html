JS Console

A developer left a debug function in MegaCorp's error page. Find and call it.

▸
Open DevTools (F12) → Console tab. Look at the page's JavaScript source (Ctrl+U or Sources tab) to find the function name, then call it from the console.

Every JavaScript file delivered to the browser is public. Debug functions, internal API endpoints, hardcoded keys, and hidden logic are all visible to anyone who opens the Sources tab. Many real breaches have started with a developer forgetting to remove a debug helper.

The lesson: There is no such thing as secret JavaScript. Everything you send to the browser can be read.
