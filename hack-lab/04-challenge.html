localStorage

MegaCorp's app stores your session token in the browser. Find where.

🎯
Open DevTools (F12), go to the Application tab (Chrome) or Storage tab (Firefox). Look under Local Storage → the page URL. One of those keys holds a secret.

localStorage is a key-value store in your browser. Open DevTools → Application tab → Local Storage → select this page's origin. You'll see all keys stored by this page, including one called megacorp_session.

🔓 Challenge solved!

localStorage is readable by any JavaScript running on the page — including injected scripts (XSS attacks). Session tokens, user IDs, and auth data stored in localStorage can be stolen if an attacker can run JS on the page. Modern apps use HttpOnly cookies instead, which JS cannot read.

The lesson: localStorage is not a safe place for authentication tokens or sensitive data.
