Website Progress

Add mandatory JavaScript features to your site: a live countdown and at least one interactive element.

📅 Deadline — 15 / 17 April

This is the technical core of the project. Your website must now demonstrate real JavaScript skills with two mandatory features.

Req. 1Countdown to Launch

Live countdown using Date() and setInterval()

Your website must display a live countdown that ticks every second, showing the time remaining until a fictional "launch date" for your startup.

  • 1
    Choose a target date
    Pick a future date for your startup's fictional launch. Store it as a Date object in your JavaScript.
  • 2
    Calculate the difference
    Subtract the current time (new Date()) from the target date to get the remaining milliseconds. Convert to days, hours, minutes and seconds.
  • 3
    Update every second
    Use setInterval(function, 1000) to re-calculate and update the display every second. The countdown must tick live on the page.
  • 4
    Display it clearly
    Show the countdown in a prominent place on the page (hero section, dedicated banner, etc.). Use document.getElementById() to update the text content.
💡
Remember Lesson 14. You practised this exact technique in the Date & Time lesson. Go back to those exercises if you need a refresher on how Date(), getTime() and setInterval() work together.

Req. 2Interactive Feature

At least one additional interactive element

Besides the countdown, your website must include at least one more interactive feature built with JavaScript. Here are some ideas:

  • Contact form with alert: a form that validates inputs and shows a confirmation message using alert() or a custom modal
  • FAQ accordion: questions that expand/collapse when clicked, using addEventListener and classList.toggle
  • Image slider / carousel: a rotating set of images with next/previous buttons
  • Newsletter sign-up: an email input that validates the format and gives feedback
  • Dark mode toggle: a button that switches between light and dark colour schemes using CSS classes
  • Scroll animations: elements that fade or slide in as you scroll down the page

Choose something that makes sense for your startup. A financial education app could have an interactive savings calculator. A commuter app could have a route planner mockup.

Using AI (ChatGPT, Claude, etc.)

🤖
AI is allowed — but understanding is mandatory. You are authorised to use AI tools to help generate and structure your code. However, during the presentation you will be asked to explain every single line of code in your project. If you used AI, it is your responsibility to study, understand, and be able to justify every piece of code you include. Points are lost if you cannot explain your own code.

Smart AI usage tips

  • Ask the AI to explain the code it generates — don't just copy-paste
  • Try to modify the output (change colours, text, layout) to prove you understand it
  • Document your AI prompts in the development diary
  • If the AI gives you code you don't understand, ask it to break it down step by step
