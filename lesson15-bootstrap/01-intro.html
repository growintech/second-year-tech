The Navbar Component

Learn what a CSS framework is and how Bootstrap's navbar works.

A CSS framework is a collection of pre-written CSS rules activated by class names.

Bootstrap provides a responsive grid, ready-made UI components, and utility classes. You load it with a single <link> tag.

The Navbar is responsive by default, collapsing on small screens and expanding on large ones.

💡
Tip Pair navbar-dark with bg-dark for a classic dark navbar.

Key Classes

Base class, required on the <nav> element. Enables flexbox layout and sets up navbar structure.

Collapses the nav on screens smaller than "lg" and expands horizontally on large screens.

navbar-dark sets links to light; bg-dark applies the background color.

Styles the logo or site-name link applied to the <a> tag.

nav-item goes on each <li> inside the nav list; nav-link goes on the <a> inside it.

container centers content; ms-auto pushes nav links to the right.

⚠
Common mistake: Without ms-auto on the <ul>, your links will stay on the left.
