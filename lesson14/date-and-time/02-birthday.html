Birthday Finder

Use getDay() to find out which day of the week you were born on. You can verify the result on Google afterwards!

Your task

Write a script that…

1

Creates a Date with your birthday

Use the format "YYYY-MM-DD". If you don't know the exact date, use your birth year and a date you remember.

2

Calls getDay() to get the day of week

This returns a number from 0 (Sunday) to 6 (Saturday).

3

Shows the result with console.log()

Which number comes out? Use the table below to find the day name.

Solution

Click to reveal — try on your own first! 
var birthday = new Date("2008-05-13");
var day = birthday.getDay();
console.log("Day of week: " + day);
// 2 = Tuesday
⭐
Bonus challenge Show a different message if the day is 0 (Sunday) or 6 (Saturday): 
if (day == 0 || day == 6) {
  console.log("You were born on a weekend!");
} else {
  console.log("You were born on a weekday.");
}
