Build it — step by step

Step A

Show only the days

Start simple. Calculate the days left and put the number into #days. Ignore hours/minutes/seconds for now.

var target = new Date("2025-07-11 14:00:00");
var now = new Date();
var diff = target.getTime() - now.getTime();
var days = parseInt(diff / (24 * 60 * 60 * 1000));

document.querySelector("#days").innerText = days;
Step B

Add hours, minutes, seconds

After extracting days, subtract them from diff, then extract hours from the remainder. Repeat for minutes and seconds.

Step C

Wrap everything in setInterval

Right now your code runs once and freezes. Move all of it inside setInterval(function() { ... }, 1000) so it recalculates every second.

💡
Key point Move var now = new Date() and var diff = ... inside the interval too — they must be recalculated each tick.

Full solution

Click to reveal — only after completing step C! 
var target = new Date("2025-07-11 14:00:00");

setInterval(function() {

  var now = new Date();
  var diff = target.getTime() - now.getTime();

  var days = parseInt(diff / (24 * 60 * 60 * 1000));
  diff = diff - days * (24 * 60 * 60 * 1000);

  var hours = parseInt(diff / (60 * 60 * 1000));
  diff = diff - hours * (60 * 60 * 1000);

  var minutes = parseInt(diff / (60 * 1000));
  diff = diff - minutes * (60 * 1000);

  var seconds = parseInt(diff / 1000);

  document.querySelector("#days").innerText = days;
  document.querySelector("#hours").innerText = hours;
  document.querySelector("#minutes").innerText = minutes;
  document.querySelector("#seconds").innerText = seconds;

}, 1000);
