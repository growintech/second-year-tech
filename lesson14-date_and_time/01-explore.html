The Date object

JavaScript has a built-in Date object for working with dates and times. Let's explore its methods — type code below and see the results live.

Step 1 — Create a date

1

Today's date and time

Calling new Date() with no arguments gives you the current moment. Run the code below and see what appears.

var now = new Date();
console.log(now);
2

A specific date

Pass a string in YYYY-MM-DD format to create a date for a specific day:

var birthday = new Date("2008-05-13");
console.log(birthday);

Step 2 — Extract values

A Date object has many methods to read individual parts. Try each one and predict the output before running. 

var now = new Date();

now.getFullYear()   // the year, e.g. 2025
now.getMonth()      // ⚠️ tricky! see below
now.getDate()       // day of the month (1–31)
now.getDay()        // ⚠️ tricky! see below
now.getHours()      // hour (0–23)
now.getMinutes()    // minutes (0–59)
now.getSeconds()    // seconds (0–59)
⚠️
Two famous gotchas getMonth() returns 0 for January, 1 for February… 11 for December.

getDay() returns 0 for Sunday, 1 for Monday… 6 for Saturday.

Questions to answer

Click each question to reveal the answer.

What does getMonth() return if today is in January?
0 — not 1. January is month 0.
What is the difference between getDate() and getDay()?
getDate() → day number in the month (e.g. 13). getDay() → day of the week (e.g. 2 = Tuesday).
If today is a Monday, what does getDay() return?
1 — because Sunday = 0, Monday = 1.
What does getFullYear() return right now?
The current 4-digit year, e.g. 2025.
