⏱️ Concept

getTime() and
time differences

To calculate how many days between two dates, we need a common unit. JavaScript measures time in milliseconds — and that's the key.

What is getTime()?

The Unix Epoch

getTime() returns the number of milliseconds since 1 January 1970, 00:00:00 UTC — the "Unix Epoch". Both dates become plain numbers, so subtracting one from the other gives the difference in milliseconds. 

var now = new Date();
console.log(now.getTime());
// e.g. → 1746000000000

Calculating a difference

1

Subtract two dates

var d1 = new Date("2025-01-01");
var d2 = new Date("2025-07-11");

var diff = d2.getTime() - d1.getTime();
console.log(diff);   // milliseconds
2

Convert milliseconds → days

1,000 ms = 1 second · 60 seconds = 1 minute · 60 minutes = 1 hour · 24 hours = 1 day:

var days = parseInt(diff / (24 * 60 * 60 * 1000));
console.log("Days between: " + days);
⚠️
Remember parseInt() Division gives a decimal like 84.07. Use parseInt() to keep only the whole number: parseInt(84.07) → 84.
