Step 1 — Navbar

Build a Bootstrap Navbar

Add a responsive navigation bar to the top of the page using Bootstrap classes.

  • Add a <nav> with classes navbar navbar-expand-lg navbar-dark bg-dark
  • Inside, add a <div class="container">
  • Add a brand link: <a class="navbar-brand" href="javascript:void(0)">Net@</a>
  • Add a <ul class="navbar-nav ms-auto"> with at least two nav-link items
  • Include the text Net@ as the brand name
Hint: Each link needs two classes: nav-item on the <li> and nav-link on the <a>.
Hint 1: A navbar uses a <nav> element. Bootstrap styles it automatically — you just add the right class names.
Hint 2: Use navbar-dark bg-dark for a dark look. navbar-expand-lg makes it collapsible on mobile.
Hint 3: Each link needs <li class="nav-item"><a class="nav-link">...</a></li>. Put the brand link before the <ul>.
