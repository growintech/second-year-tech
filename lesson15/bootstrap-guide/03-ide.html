Step 3 — Grid Layout

Add a Sidebar Layout

Use Bootstrap's 12-column grid to create a two-column layout: a main content area and a sidebar.

  • Add a <div class="container py-5"> section
  • Inside, add a <div class="row">
  • Add a main column: <div class="col-12 col-lg-8">
  • Add a sidebar column: <div class="col-12 col-lg-4">
  • The two columns must use col-lg- responsive breakpoints
Hint: Column numbers must add up to 12. col-lg-8 + col-lg-4 = 12. On mobile (col-12), both take full width and stack.
