Step 5 — Footer

Add a Footer

Complete your landing page with a footer. Bootstrap has no footer component — compose utility classes on a <footer> element.

  • Add a <footer> element
  • Give it a dark background using bg-dark directly on the <footer> tag
  • Add text-white, py-4, and mt-5
  • Inside, add a container text-center div
  • Include the text Net@ in the footer
  • Include a copyright notice with the year 2025 and the © symbol
Hint: The © symbol in HTML is &copy; or the character © pasted directly. text-muted on the paragraph gives it a softer grey color.
Hint 1: Use the semantic <footer> HTML element. Bootstrap has no footer component — you compose utility classes directly on it.
Hint 2: Stack the classes on the footer element: class="bg-dark text-white py-4 mt-5". py-4 adds top/bottom padding, mt-5 adds margin-top.
Hint 3: Inside the footer, add <div class="container text-center"> then a paragraph with the © symbol: you can type © directly or use &copy;.
