Step 2 — Hero Section

Build a Hero Section

Add a big, bold hero section below the navbar. The hero is the first thing visitors see — make it count.

  • Add a <section> with py-5 text-center (and optionally a background color like bg-primary text-white)
  • Inside, add a <div class="container">
  • Add a headline with display-4 (or higher like display-3) class
  • Add a subtitle paragraph with the lead class
  • Add at least one btn button
  • Include the text Net@ somewhere in the hero
Hint: display-4 makes an <h1> very large. lead makes a <p> slightly bigger and lighter — perfect for hero subtitles.
Hint 1: A hero section is usually a <section>. Use utility classes to control spacing and alignment.
Hint 2: py-5 adds vertical padding. text-center centres everything. display-4 makes a big headline.
Hint 3: lead gives you a larger, lighter paragraph. Add a .btn element for the CTA button. Any colour variant works.
